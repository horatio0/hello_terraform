# cdn이랑 잘쓰면 정적 웹 서버 ( nginx 같은 정적파일 제공 서버 ) 로 사용 가능

# aws s3 cp <복사할 파일 이름> s3://<S3 버킷 이름>/                 => 업로드 명령어

[IAM]
- IAM User : 사용자
- IAM Group : 사용자( IAM User )의 집합
- IAM Policy : 권한
- IAM Role : 특정 개체( User, 다른 AWS 서비스 )에 부여하는 권한 ( IAM Role )의 집합
